# Unit Economics

## Costs per Audit

1. **Groq API Inference (`llama-3.3-70b-versatile`)**
   - Cost: ~$0.59 per 1M tokens.
   - Average prompt + response = ~800 tokens.
   - **Cost per run:** ~$0.00047

2. **Supabase Hosting**
   - Free tier includes 500MB DB space, which is enough for ~100,000 audits.
   - Marginal cost: **$0.00**

3. **Resend Emails**
   - Free tier includes 3,000 emails/month.
   - **Cost per lead capture:** $0.00 (until scale)

4. **Vercel Hosting**
   - Serverless compute + Edge network.
   - Free tier is sufficient for initial launch.

**Total COGS per audit:** ~$0.0005

## Expected Value

- **Conversion Rate:** Assuming 5% of users who complete the audit leave an email.
- **Consultation Value:** If 1 in 100 leads converts to a paid consultation at $500.
- **Expected Value per Lead:** $5.00
- **Expected Value per Audit:** $0.25

**Margin:** The unit economics are highly favorable, allowing us to run the tool for free indefinitely as a lead generation magnet.
